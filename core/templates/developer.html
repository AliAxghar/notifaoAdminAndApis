Create App
POST createapp/Expand

Create a new Notifao App.

Body Parameters

email required
Type String
It is used for login.
password required
Type String
It is used for login.
app_name required
Type String
It is used for your application name.
app_discription required
Type String
It is used for your application description.
app_image required
Type File
It is used for your application image.
app_logo required
Type File
It is used for your application logo.

Json Response

{
     "id": 61,
    "name": "string",
    "description": "string",
    "notifications_used": 0,
    "app_qr": QR_imge.png    ,
    "created_at": "2021-01-09",
    "app_image": "image.jpg",
    "status": 200
}
